# Miguel Viegas | Portfolio Website

**Delivering complex technology initiatives through structured execution, stakeholder alignment, and reliable delivery**

Personal portfolio website showcasing my experience, selected projects, and professional background in project management, product delivery, and technology-led change.

## About

This website was created to present my professional profile in a clear, structured, and accessible format. It brings together my experience delivering complex technology initiatives, leading cross-functional teams, and driving business impact through effective planning, governance, and collaboration.

The site is designed primarily for recruiters, hiring managers, and collaborators looking for a concise overview of my experience, supported by selected project highlights and a downloadable CV.

## Features

- Professional summary and profile  
- Key skills
- Selected projects with key contributions
- Career experience timeline    
- Education  
- Certifications 
- Downloadable CV  
- Clean, responsive layout for desktop and mobile  

## Tech Stack

- HTML5  
- CSS3  
- JavaScript (vanilla)  
- GitHub Pages (for hosting)  

## Design Approach

The website follows a minimal and monochromatic design to keep the focus on content and readability.

Key principles:

- Clarity over visual complexity  
- Easy scanning for recruiters  
- Consistent layout and spacing  
- Lightweight and fast-loading  
- Mobile-first considerations
